# **Uptute - Educational Platform**

## **Project Overview**

**Uptute** is an online educational platform that offers a wide range of courses for students of various levels. The platform provides interactive learning tools and resources for subjects such as Computer Science, IT, and coding. The website is designed with a clean and modern interface, offering easy navigation between different sections like the homepage, course offerings, about page, and a contact form.

This repository contains the full front-end code for **Uptute**, including all HTML, CSS, and JavaScript files needed to run the website. The platform is built to provide a smooth user experience for visitors to learn more about the platform, browse courses, and get in touch with the support team.

---

## **Features**

### **1. Home Page**
- **Hero Section**: The main introductory section that provides a quick overview of what Uptute offers.
- **About Us**: A brief description of the company’s mission and vision.
- **Courses Overview**: Featured courses with links to more details on the Courses page.
- **Call to Action**: Directs users to contact or learn more about specific offerings.

### **2. About Page**
- Provides detailed information about the platform, its services, mission, and core values.
- Includes a section to help users understand what sets Uptute apart from other educational platforms.

### **3. Courses Page**
- A catalog of courses offered by Uptute, such as:
  - Cambridge IGCSE ICT & Computer Science
  - AQA GCSE and A Level Computer Science
  - OCR A Level Computer Science
  - Python coding for grades 6-8, AI tools for school students, and more.
  
Each course includes:
- Course title
- Short description
- Course image
- A link to get more information through the contact form

- **Load More Button**: Loads additional courses to display as the user scrolls down.

## **Technologies Used**
- **HTML5**: The structure and content of the pages.
- **CSS3**: Custom styles for layout, fonts, and responsive design.
- **JavaScript**: For client-side interactivity, including form validation and custom alerts.
  - **Form Validation**: Ensures correct data input for login and registration forms.
  - **Custom Alerts**: Provides feedback to users on successful or failed actions.
- **Font Awesome**: Icons for social media and contact methods (e.g., WhatsApp, Email).
- **Swiper.js**: Potential integration for smooth scrolling (though not fully implemented in the current contact page).
- **Formspree**: Simple form backend for submitting contact messages.

### **Prerequisites**
- A browser to view the website.
- A text editor to view or modify the code (e.g., VS Code).

   ```bash
